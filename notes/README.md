# StudyShift 🌍

A global student work marketplace powered by Stellar Soroban escrow.

---

## Problem

College students in Southeast Asia struggle to find safe, flexible, and well-paying online work. Many lose income due to delayed or missing payments from informal employers.

---

## Solution

StudyShift connects students with global employers using Soroban smart contract escrow. Employers lock funds before work starts, and payments are released instantly upon approval.

---

## Stellar Features Used

- Soroban Smart Contracts
- USDC Token Transfers
- On-chain Escrow
- Instant Settlement

---

## Architecture

Frontend: React / Next.js  
Wallet: Freighter / Stellar Wallet  
Backend: Soroban Contract  
On-chain: Job state + escrow funds
